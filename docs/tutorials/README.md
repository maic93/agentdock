# Tutorials

Task-oriented, runnable walkthroughs ("build your first plugin," "add a
workflow template"), tested in CI rather than written once and left to rot.

## Current status

Empty. The first tutorials will be written once there is something real to
walk through — most likely "run the Phase 0 vertical slice locally" once
the first kernel and foundation packages exist (see the roadmap in
[docs/architecture/001-system-architecture.md](../architecture/001-system-architecture.md#24-future-roadmap-indicative-post-core)).

Writing a tutorial against not-yet-existing functionality would produce
exactly the kind of "coming soon" placeholder content this repository's
foundation phase explicitly avoids.
