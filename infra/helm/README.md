# infra/helm

Kubernetes deployment charts, for managed/cluster deployments beyond the
Docker Compose self-host path in `infra/docker/`.

## Status

Deferred. Per the future-improvements list in
`docs/architecture/002-repository-foundation.md`, Section 13, Helm chart
work is intentionally deferred until there's real demand for managed/
Kubernetes deployments — building this now, with no services to deploy yet,
would be speculative infrastructure with nothing to validate it against.
